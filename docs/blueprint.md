# **App Name**: UNDB ProfAssist

## Core Features:

- Institutional Login: Secure access using @undb.edu.br email validation.
- Course Management: Add, edit, archive, and delete courses.
- AI-Powered Lesson Plan Import: Extract data from PDF lesson plans using Gemini to automatically create course structures. Allows instructor to review the extracted information and approve with confirmation, so the extraction of this AI feature works like a tool for the professor to double-check
- Classroom Management: Manage different class groups, define grading rules, and associate with courses.
- Student Management: Import student lists via CSV, edit individual student details, and group students.
- Document Storage: Store reports, spreadsheets, and PDFs associated with courses and classes.
- Academic Calendar: Display important academic events with visual integration by courses.

## Style Guidelines:

- Primary color: Medium purple (#9466DE) to align with UNDB's brand identity, offering a modern and institutional feel.
- Background color: Light purple (#F1EAF9), offering a clean, unobtrusive canvas
- Accent color: Deep blue (#4A77D4) for CTAs and interactive elements, providing visual distinction and guiding user interaction.
- Body and headline font: 'PT Sans', a versatile sans-serif for clear and accessible information presentation.
- Intuitive icons to enhance navigation and clarify functionalities.
- Clean, card-based layout with prominent buttons and clear visual feedback.
- Subtle transitions and animations for smooth navigation and user interaction.
