# UNDBProf

Esta é uma ferramenta de auxílio pessoal, criada por um professor para professores, com o objetivo de otimizar e automatizar tarefas acadêmicas.

**Importante:** Este aplicativo não possui nenhum vínculo oficial com a instituição UNDB e não substitui os sistemas oficiais da faculdade. É um projeto independente para facilitar a gestão de turmas e disciplinas.

Para começar, explore a aplicação após o login. A página de Dashboard contém um tutorial rápido para os primeiros passos.
